# Thanatos
An internal C++ skinchanger featuring sdk and hooks.

## Hooks
- FrameStageNotify
- CreateMove

## Interfaces
- IEngineClient
- IBaseClientDLL
- IModelInfoClient
- IClientEntityList
- ICvar

## Config
To use it place the "Thanatos.ini" config in your csgo folder and press "end" to eject.
PS: The cheat will crash if no config is found.
