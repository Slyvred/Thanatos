# Thanatos
An internal C++ skinchanger featuring sdk and hooks.

## Hooks
-FrameStageNotify: skinchanger
-CreateMove: rank revealer

## Interfaces
-IEngineClient
-IBaseClientDLL
-IModelInfoClient
-IClientEntityList
-ICvar

## Config
To use it place the "Thanatos.ini" config in your csgo folder and press "end" to eject.
